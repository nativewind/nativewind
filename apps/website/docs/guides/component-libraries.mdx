# Component Libraries

Here are some popular component libraries that work well with NativeWind:

## [NativeWindUI](https://nativewindui.com)

A collection of beautifully designed, accessible components built specifically for NativeWind. Provides a comprehensive set of UI primitives that work seamlessly across platforms while maintaining native feel.

## [React Native Reusables](https://github.com/mrzachnugent/react-native-reusables)

A lightweight library of essential React Native components, inspired by shadcn/ui, styled with NativeWind. Focuses on commonly needed UI patterns with minimal overhead and maximum flexibility.

## [gluestack](https://gluestack.io/)

An extensive universal component library that supports NativeWind styling. Offers a rich ecosystem of customizable components that work across React Native and web platforms with consistent behavior.

Each of these libraries provides different approaches to component development while leveraging NativeWind's styling capabilities. Choose the one that best fits your project's needs and design requirements.
